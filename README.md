# Парсер словаря

Словарь можно скачать с сайта http://www.speakrus.ru/

Опробовал на словаре: Орфографический словарь п/р проф. Лопатина (2000 год). Краткий вариант без ударений.

Словарь необходимо сохранить в кодировке UTF-8
